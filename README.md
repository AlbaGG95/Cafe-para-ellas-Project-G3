☕ Café para Ellas

Web estática centrada en una experiencia visual elegante y acogedora inspirada en el mundo del café.

🧩 Descripción

Café para Ellas es un proyecto frontend desarrollado únicamente con HTML y CSS, enfocado en la maquetación web, el diseño visual y la creación de interfaces limpias y atractivas.

El objetivo del proyecto es practicar la estructura semántica del HTML y el diseño responsive con CSS, cuidando la estética y la experiencia de usuario.

🚀 Tecnologías utilizadas
🧱 HTML5
🎨 CSS3
📐 Diseño responsive
🧩 Layout con Flexbox / Grid
📂 Estructura del proyecto
project/
│── index.html
│── style.css
│── assets/        # Imágenes y recursos
🌐 Funcionalidades
Diseño visual atractivo inspirado en cafeterías ☕
Estructura semántica HTML
Layout responsive adaptable a diferentes dispositivos
Uso de Flexbox/Grid para organización del contenido
Separación de estructura (HTML) y estilos (CSS)
⚙️ Cómo usar el proyecto
Clona el repositorio:
git clone https://github.com/AlbaGG95/Cafe-para-ellas-Project-G3.git
Abre el proyecto:
cd Cafe-para-ellas-Project-G3
Ejecuta abriendo el archivo en tu navegador:
index.html

Practicar maquetación web desde cero
